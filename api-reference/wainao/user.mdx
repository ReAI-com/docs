# 用户管理

## 获取用户信息

<ParamField path="/user" method="GET">
获取当前登录用户的详细信息。

### 响应格式

```json
{
  "code": 200,
  "data": {
    "id": "string",
    "name": "string",
    "avatar": "string",
    "createdAt": "string",
    "updatedAt": "string"
  },
  "message": "获取成功"
}
```

### 错误码说明

| 错误码 | 说明 |
|--------|------|
| 401    | 未授权 |
| 500    | 服务器错误 |

</ParamField>

## 更新用户信息

<ParamField path="/user" method="PUT">
更新当前登录用户的信息。

### 请求参数

```json
{
  "name": "string",    // 用户名称
  "avatar": "string"   // 头像URL
}
```

### 响应格式

```json
{
  "code": 200,
  "data": {
    "id": "string",
    "name": "string",
    "avatar": "string",
    "updatedAt": "string"
  },
  "message": "更新成功"
}
```

### 错误码说明

| 错误码 | 说明 |
|--------|------|
| 400    | 请求参数错误 |
| 401    | 未授权 |
| 500    | 服务器错误 |

</ParamField>
